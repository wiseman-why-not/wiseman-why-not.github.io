Cast a double to an int

What is "cast a double to an int"

Cast a double to an int can better be said as convert a double to and integer.

What is casting? Casting is taking an Object of specific type and making it into another Object type. Converting a numberic value from one type to another type.

Casing is a process of converting one type to another, but in Java, types from teh same type hierarchy can be cast or converted into each other. If you try to cast two objects which don't share the same type hierarchy, i.e there is no parent-child relationshiop between them. You will get a complie time error.

Why cast a double to an int

You could be dealing with a method that returns a double, but you might not want the numbers after the decimal or need to change it to an int so you can use it with another method that uses int values. Casting a double to and int will remove the numbers after the decimal, it will not round up.

What does it look like

double myDoubleNumber = 45.12;

int myIntNumber = (int)myDoubleNumber;

myIntNumber = 45


Double myDoubleNumber = 45.12;

int myIntNumber = Integer.valueOf(myDoubleNumber.intValue() );

myIntNumber = 45

How is this happening?

double and int are primitive data types that start with a lowercase character. Double and Integers are numeric classes, which makes it possible to use them like objects and using methods on method like inValue() or valueOf().
