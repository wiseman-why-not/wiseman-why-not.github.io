Parameter or Argument

What is a Parameter?

A parameter is a variable defined in a method declaration. When we define a method, we introduce our compiler with ome variables that are being used in the running of that method. These variables are the parameters.

What is an Argument?

An Argument is the actual value or data being passed in the method, when calling upon the method. when defining a method, you create variables or parameters, but when calling or using the method, you need to replace the variables with data, or arguments.

Visuals please!

public void sum(int xParameter, int yParameter){

return xParameter + yParameter;

}

int xArgument = 5;

int yArgument = 15;

myClass.sum(xArgument, yArgument);

We can see here that the parameters are defined when creating a function, and Arguments are the actual data we are passing in the method.
